============================================================
[WEB-24] 웹서비스관리자페이지노출제한
============================================================
[WEB-24-START]

관리자 페이지 location 블록이 발견되지 않았습니다. 노출 위험 낮음.

[현황]
1) 진단 확인
command: grep -E '^\s*location.*/(admin|administrator|wp-admin|phpmyadmin)' /etc/nginx/nginx.conf /etc/nginx/conf.d/*.conf 2>/dev/null | grep -v '^\s*#'
command_result:
No admin locations found

[WEB-24-END]

[WEB-24]Result : GOOD

[참고]
진단 목적: 관리자 페이지 접근 제한으로 무단 접속 방지
보안 위협: 관리자 페이지 노출 시 무단 접속 및 시스템 장악 위험
양호 기준: 관리자 페이지에 접근 제한이 설정된 경우
취약 기준: 접근 제한이 없는 경우
조치 방법: location /admin { allow x.x.x.x; deny all; } 또는 auth_basic 설정
